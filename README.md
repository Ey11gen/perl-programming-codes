## Running the Examples

```bash
perl script_name.pl
```

For chapters requiring CPAN modules, install dependencies:
```bash
cpanm Module::Name
```

## Requirements

- Perl 5.26 or later (Perl 5.38 recommended)
- CPAN modules as noted per chapter
- Linux, macOS, or Windows with Strawberry Perl

## Author Notes

All examples use `use strict;` and `use warnings;` as professional
best practice. Code is organized to match the book chapter by chapter.
